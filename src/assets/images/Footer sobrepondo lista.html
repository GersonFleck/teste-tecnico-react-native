O ChatGPT disse:

Ótimo — sua imagem mostra que o footer com o texto “NBA Stats App” está em cima do final da lista do FlatList, cobrindo o último nome.

Isso está acontecendo porque você usou:

jsx
position: 'absolute',
bottom: 0,

no seu <Footer />, mas não deixou “espaço” para ele no layout geral. Assim, a lista vai até o fim da tela — e o footer fica por cima dela.

✅ Para corrigir isso de forma padrão no React Native, você tem duas opções principais:

✅ 1️⃣ Opção mais simples: adicionar padding ao FlatList

Adicione contentContainerStyle ao seu FlatList com paddingBottom igual ou maior à altura do footer:

jsx
<FlatList
  contentContainerStyle={{ paddingBottom: 30 }}
  ...
/>

✅ Assim, o último item da lista não ficará escondido.

✅ 2️⃣ Melhor ainda: usar Layout com flex

Em vez de colocar o Footer absoluto, use um layout com flex:

jsx
<View style={{ flex: 1 }}>
  <FlatList
    ...
  />
  <Footer />
</View>

E no Footer, tire o position: 'absolute':

jsx
const footerStyles = StyleSheet.create({
  container: {
    height: 30,
    backgroundColor: '#616161',
    justifyContent: 'center',
    alignItems: 'center',
  },
  text: {
    color: 'white',
    fontSize: 16,
  },
});

✅ Resultado: o Footer sempre ficará fixo embaixo, sem sobrepor nada.
